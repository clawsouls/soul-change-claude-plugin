# Soul Change — Claude Plugin

Give your AI a soul. Browse 80+ AI personas from [ClawSouls](https://clawsouls.ai) and apply them instantly.

## Install

In Claude Code:
```
/install-plugin https://github.com/clawsouls/soul-change-claude-plugin
```

## What it does

- **Search** 80+ curated AI personas by category or keyword
- **Preview** personality files (SOUL.md, IDENTITY.md, AGENTS.md) before applying
- **Apply** a persona to transform Claude's behavior with one command
- **Install** persona packages to your local project

## Categories

Coding, Creative Writing, Education, Business, Research, Gaming, and more.
